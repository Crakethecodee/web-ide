вҡЎ Web IDE

A browser-based Web IDE that lets you write, run, and manage full-stack Python projects (Django/Flask/FastAPI) directly in the browser.

 Features

- рҹ“Ғ File Explorer вҖ” create projects, navigate files, clone GitHub repos
- рҹ“қ Code Editor вҖ” Monaco editor (VSCode's editor) with syntax highlighting
- в–¶пёҸ Process Runner вҖ” run Python servers, view live logs
- рҹ’» Interactive Terminal вҖ” PowerShell terminal in browser
- рҹ—ғпёҸ Database Viewer вҖ” browse SQLite tables, rows, run queries

Tech Stack

| Layer     | Tech                                    |
| --------- | --------------------------------------- |
| Frontend  | React + Vite + Monaco Editor + Xterm.js |
| Backend   | FastAPI (Python)                        |
| Terminal  | WebSocket + PowerShell                  |
| DB Viewer | SQLite                                  |
| Storage   | Local disk (workspace folder)           |

Setup & Run

Prerequisites

- Python 3.10+
- Node.js 18+
- Git

Backend

bash
cd backend
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
uvicorn main:app --reload --port 8000


Frontend

bash
cd frontend
npm install
npm run dev


Open `http://localhost:5173`

Usage

1. Type a project name вҶ’ click **+ New** to create a project
2. Or paste a GitHub URL вҶ’ click **Clone**
3. Click files in explorer to edit in Monaco editor
4. Go to **LOGS** tab вҶ’ enter project path + command вҶ’ click **в–¶ Run**
5. Go to **TERMINAL** tab вҶ’ type commands вҶ’ press Enter
6. Go to **DATABASE** tab вҶ’ enter SQLite path вҶ’ click **Connect**
